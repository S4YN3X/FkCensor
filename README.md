# FkCensor (Обход цензуры Яндекс Музыка)

Патчер Яндекс Музыки, который автоматически заменяет недоступные треки на рабочие версии.

## Как это работает

FkCensor модифицирует файл `app.asar` Яндекс Музыки, встраивая скрипт замены треков прямо в приложение. После установки патча никакие дополнительные программы держать запущенными не нужно - замена работает внутри самой ЯМ.

Список заменяемых треков загружается автоматически из [FckCensorData](https://github.com/Hazzz895/FckCensorData) и обновляется каждые 10 минут.

## Установка

1. Скачай последний релиз из [Releases](https://github.com/S4YN3X/FkCensor/releases)
2. Запусти `FkCensor.exe` - появится иконка в трее
3. Нажми на иконку → **Установить ASAR патч**
4. Яндекс Музыка перезапустится автоматически

> ⚠️ Яндекс Музыка должна быть установлена по стандартному пути (`%LOCALAPPDATA%\Programs\YandexMusic`)

## Удаление патча

Нажми на иконку в трее → **Удалить ASAR патч** → Яндекс Музыка перезапустится с оригинальными файлами.

## Автозапуск

В меню трея можно включить **Автозапуск с Windows** - FkCensor будет стартовать вместе с системой. При этом сам патч уже встроен в ЯМ и работает без запущенного FkCensor.

## Частые вопросы

**Яндекс Музыка не запускается после патча**  
Нажми **Удалить ASAR патч** - это восстановит оригинальные файлы. Затем переустанови патч заново.

**Трек всё равно недоступен**  
Трек может ещё не быть в базе замен. База пополняется - проверь позже.

**Обновилась Яндекс Музыка и патч слетел**  
После обновления ЯМ нужно переустановить патч через меню трея.
